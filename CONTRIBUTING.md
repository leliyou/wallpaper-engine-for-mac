# Contributing

Thanks for your interest in improving this project.

## Development Setup

1. Use macOS 12+ on Apple Silicon.
2. Install Xcode Command Line Tools.
3. Run:

```bash
swift run
```

## Branch and Commit

1. Create a feature branch: `feature/<topic>`.
2. Keep commits small and focused.
3. Use clear commit messages (e.g. `feat: add display diagnostics refresh`).

## Pull Requests

1. Explain what changed and why.
2. Include manual test steps.
3. Add screenshots or short clips for UI-related changes when possible.

## Code Style

- Follow existing Swift naming and formatting style.
- Prefer small, testable units.
- Avoid unrelated refactors in the same PR.
